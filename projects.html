EnergyCast Poland

Time Series • PyTorch

Electricity consumption forecasting for the next 24 hours using an ANN model trained on historical energy + weather features. Clean modular pipeline: preprocessing → training → inference.

Key features
  • Sliding window: 30 days → 24h forecast
  • Normalization via MinMaxScaler
  • PyTorch ANN (512 → 256 → 24)
  • Inference pipeline + plots
  • CSV + optional binary export
Tech stack
  • Python, PyTorch
  • Pandas, NumPy
  • Scikit-learn, Joblib
  • Matplotlib
  • Git/GitHub
